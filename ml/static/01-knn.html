from sklearn.neighbors import KNeighborsClassifier
from sklearn.model_selection import cross_val_score, cross_val_predict, LeaveOneOut

# first prepare our features and target
X = df.drop('target', axis='columns')
y = df['target']

# select our cross validation iterator
cvi = LeaveOneOut()

# model selection, determine the best k-neighbors value for the set
scores = []
ks = [x for x in range(0,30) if(x%2!=0)]
for k in ks:
    # create classifier
    knn = KNeighborsClassifier(n_neighbors=k)
    # cross validation with 5 folds
    a = cross_val_score(knn, X, y, cv=cvi)
    scores.append(a)

Plotting the results to see for which K, KNN works the best.

Let's now redo the previous steps with a drastic feature selection. The choice of features is done by evaluating the features' capability of being a good class fingerprint/mark, i.e. when the datas are characteristics of a class. We can try to redure our 13 dimensions matrix into a 4 dimensions one in regards to the curse of dimensionality (low number of samples implies low number of features).

To achieve this, we can compare the variance of each feature, since a high variance implies that the data points are spread out, reducing the chances that the datas are overlapping. We will pick the first three.

The last feature I have chosen is the color_intensity. I discovered its impact on the datas while attempting to write the first version of this notebook. I'll explain why it is good for us later on.
